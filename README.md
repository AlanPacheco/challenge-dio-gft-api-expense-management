# Desafio Prático do Bootcamp DIO GFT Start #4 Java :coffee:

## Sobre o projeto

É uma api web construída durante o desafio GFT Start #4, evento organizado pela [DIO](https://www.dio.me/) "Site da DIO".

O projeto consiste basicamente no gerenciamento de custos e despesas, ou seja, inicialmente formado por quatro classes e um enumeration, o usuario pode informar o valor financeiro do gasto, a categoria e quais formas de pagamento.

Obs: Pretendo futuramente evoluir esta idéia e construir um frontend para utilização real da api.

## Motivação

Com a chegada do meu segundo filho, as coisas estreitaram e exigiram ainda mais controle das finanças...kkk

## Tecnologias utilizadas
### Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- H2 Database
